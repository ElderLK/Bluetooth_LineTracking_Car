## Requirements

   - Arduino
   - L293D Shield
   - 2 Sensor TCRT5000 
   - 2 DC Motor (3-6 V)
   - HC 06 Bluetooth Module
   - Resistor 1kΩ and 2kΩ
   - Battery 5-12v
   - <a href="https://play.google.com/store/apps/details?id=braulio.calle.bluetoothRCcontroller">App Arduino Bluetooth RC Car</a>
